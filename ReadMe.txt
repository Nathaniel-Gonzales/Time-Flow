Time Flow v.1.0

Installation Instructions:
 - Install Android Studio version 2024.2.1
 - Move Time Flow folder next to this file into project directory
 - Open Time Flow as project

Emulator Instructions:
 - Set up virtual emulator in device manager
 - Use Medium Phone base 6.4" 1080x2400
 - Under System Image select API 35 "VanillaIceCream"

Running Instructions:
 - Start virtual emulator created earlier
 - Run application using the run button on top right of screen
 - Build Gradle will run and virtual emulator will then display the application
